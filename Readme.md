Render Message
=========

A small library that render message from component

## Installation

  `npm install @pparashar/testpparashar`

## Usage
npm i testpparashar
 printMessage()

 ## CODE in REACTJS
  import request, { printMessage } from 'testpparashar';
class PraveenTest extends React.Component{
    render() {
        return <h2>{printMessage()}</h2>;
    
    }
}
